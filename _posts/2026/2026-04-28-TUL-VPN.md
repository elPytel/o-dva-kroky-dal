# VPN

Pro přístup do intranetu TUL (gitlab a podobně) potřebujete klienta pro připojení přes VPN (virtuální privátní síť). 

Na Windows se doporučuje použít klienta od firmy Cisco.Cisco nabízí verzi klienta i pro Linux.

## Cisco Secure Client 

Spuštění grafické verze přes terminál:
```bash
gtk-launch com.cisco.secureclient.gui
/opt/cisco/secureclient/bin/vpnui: error while loading shared libraries: libxml2.so.2: cannot open shared object file: No such file or directory
```

{: .warning }
> Cisco Secure Client 5.1.3.62 New Features <br>
> This release includes the following features and support updates, and resolves the defects described in [Cisco Secure Client 5.1.3.62](https://www.cisco.com/c/en/us/td/docs/security/vpn_client/anyconnect/Cisco-Secure-Client-5/release/notes/release-notes-cisco-secure-client-5-1.html#secure-client-5-1-3-xx-resolved-caveats).
> - macOS 11 is no longer supported.
 > - This 5.1.3.62 release will not support Ubuntu 24.04 LTS.

Pokud při instalaci a konfiguraci na razíte na stejný problém, vyzkoušejte balík Openconnect.
## Openconnect 
Pro přístup do interní sítě TUL lze využít také klienta Openconnect.
- [openconnect](https://www.infradead.org/openconnect/)

Otestováno na: 
- Ubuntu 25.10

### GUI
Pokud chcete aplikaci Integrovat v GUI systému, včetně rychlého přepínače na systémové liště nainstalujte následující balíky.

```bash
sudo apt-get install network-manager-openconnect network-manager-openconnect-gnome
```

### Konfigurace

![nastavení_menu]({{ site.baseurl }}/assets/VPN/nastavení_menu.png){: .small .left}
*Nastavení: menu*

Přejděte do: Nastavení -> síť -> VPN

![Nastavení_VPN]({{ site.baseurl }}/assets/VPN/Nastavení_VPN.png)
*Nastavení: VPN*

Klikněte na ikonu **+** u pole **VPN**

![Nastavení_VPN_konfigurace]({{ site.baseurl }}/assets/VPN/Nastavení_VPN_konfigurace.png)
*Nastavení: VPN - konfigurace*

Přejděte do záložky: **Identita** a vyplňte následujíc položky:

| Položka      | Hodnota                           | Význam                              |
| ------------ | --------------------------------- | ----------------------------------- |
| Název        | TUL                               | Informace pro vás                   |
| VPN Protocol | Cisco AnyConnect nebo OpenConnect | Protokol kompatibilní se školní VPN |
| Gateway      | vpn.tul.cz                        | FQDN školního serveru               |

**Uložit**

Při připojení se vás klient zeptá na:
- email,
- heslo pro vzdálený přístup.

{: .note } 
> Možná chyba: `404` <br>
> Pokud má grafická konfigurace problém, použijte pro první spouštění příkazový řádek.

### Terminál

```bash
openconnect --protocol=anyconnect vpn.tul.cz
```
- `--protocol=anyconnect` - napojení na cisco protokol

Interaktivně se vás vyptá na:
- TUL/TUL_only,
- email,
- heslo.

{: .note }
> Tip: <br>
> Po prvotním připojení z terminálu se obvykle chytne i grafická varianta.
